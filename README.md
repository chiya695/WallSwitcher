# 壁纸切换器 (WallSwitcher)

一个简单的 Android 壁纸切换应用，可以自动或手动切换壁纸。

## 功能

- 从指定文件夹加载壁纸
- 自动切换壁纸（后台服务）
- 手动切换壁纸（桌面小部件）
- 随机或顺序切换
- 设置切换间隔（分钟/小时/天）
- 同时设置锁屏壁纸
- 多种裁剪方式（不裁剪/居中/顶部/底部）
- 图片质量设置（高质量/平衡/省内存/禁用压缩）
- 日志记录

## 技术栈

- Kotlin + Jetpack Compose (Material 3)
- Room + DataStore
- Coroutines + MVVM
- Navigation Compose
- WorkManager
- Coil

## 系统要求

- 最低：Android 7.0 (API 24)
- 目标：Android 16 (API 36)

## 项目结构

- `data/` — 数据模型、DAO、仓库
- `service/` — 壁纸切换服务
- `ui/` — 界面和视图模型
- `util/` — 文件、壁纸、日志等工具类
- `widget/` — 桌面小部件
- `receiver/` — 开机启动接收器

## 使用方法

1. 在主页选择壁纸文件夹
2. 在设置页面配置切换方式、顺序、间隔等
3. 自动切换就点"启动服务"，手动切换就加个桌面小部件

## 权限

- 读取外部存储 — 访问壁纸文件
- 设置壁纸 — 更改系统壁纸
- 接收开机广播 — 开机自动启动服务
- 前台服务 — 保持壁纸切换服务运行
