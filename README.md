# 壁纸切换器 (WallSwitcher)

一个简单的Android壁纸切换应用，可以自动或手动切换壁纸。

## 功能特点

- 支持从指定文件夹加载壁纸
- 支持自动切换壁纸（后台服务）
- 支持手动切换壁纸（桌面小部件）
- 支持随机或顺序切换壁纸
- 支持设置切换间隔（分钟/小时/天）
- 支持同时设置锁屏壁纸
- 支持多种壁纸裁剪方式（居中/顶部/底部）
- 支持日志记录

## 技术栈

- Kotlin：主要开发语言
- Jetpack Compose：声明式UI框架
- Room：数据持久化
- DataStore：轻量级数据存储
- Coroutines：异步编程
- MVVM架构：视图模型分离

## 项目结构

- `data`：数据层，包含数据模型、DAO、仓库等
- `service`：服务层，包含壁纸切换服务
- `ui`：UI层，包含界面和视图模型
- `util`：工具类，包含文件、壁纸、日志等工具
- `widget`：小部件，包含桌面小部件
- `receiver`：广播接收器，包含开机启动接收器

## 使用方法

1. 在主页选择壁纸文件夹
2. 在设置页面配置切换方式、顺序、间隔等
3. 如果选择自动切换，点击"启动服务"按钮
4. 如果选择手动切换，添加桌面小部件并点击切换

## 权限说明

- 读取外部存储：用于访问壁纸文件
- 设置壁纸：用于更改系统壁纸
- 接收开机广播：用于开机自动启动服务
- 前台服务：用于保持壁纸切换服务运行
